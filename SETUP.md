# Setup Instructions

The Agent created the full project structure. Please run the following commands to initialize the system:

## 1. Backend Setup (Node.js + MongoDB)
Open a terminal in the `backend` folder:
```bash
cd backend
npm install
npm start
```
You should see: `âœ… MongoDB Connected: campusplacement`

## 2. Frontend Setup (React + Tailwind)
Open a new terminal in the `frontend` folder:
```bash
cd frontend
npm install
npm run dev
```
Open the provided Local URL (e.g., http://localhost:5173) to see the application.

The `ai-agents/agent_runner.py` is ready for future logic.
